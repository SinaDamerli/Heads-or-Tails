# Heads or Tails Game

This is a simple Heads or Tails game where the player tries to guess the outcome of a coin flip. The first player (either the user or the computer) to reach a score of 10 wins the game. After the game ends, the player can restart by making another guess.

## Table of Contents
1. Installation
2. Usage
3. Features

---

## Installation

No installation is required. The project runs directly in your web browser.

### Prerequisites

- A modern web browser (Google Chrome is recommended).

## Usage

1. Open the index.html file in your browser.
2. Start the game by choosing either "Heads" or "Tails".
3. After each guess:
   - If your guess matches the computerâ€™s choice, you win the round and get a point.
   - If your guess does not match, the computer wins the round and gets a point.
4. The game continues until either the player or the computer reaches 10 points.
5. When a player wins, the game will display the winner.
6. To restart the game, simply choose "Heads" or "Tails" again. The game will reset and start a new round.

## Features

- Score Tracking: Tracks the score of both the player and the computer.
- Automatic Restart: After a game ends, the player can restart the game by making another guess.
- First to 10 Wins: The game ends when either the player or the computer reaches a score of 10.
